# FloatChat 🌊 - Conversational AI for ARGO Ocean Data Explorer

**FloatChat** is a conversational AI system designed to solve the **Smart India Hackathon Problem Statement SIH25040** (Ministry of Earth Sciences). It bridges the gap between complex raw NetCDF oceanographic datasets generated by global autonomous ARGO floats and non-expert users (students, policymakers, maritime operators) by allowing them to query ocean data in plain English.

---

## 🚀 Key Features & Differentiators

1. **Robust Text-to-SQL Translation**: Translates complex natural language prompts (e.g. *"Show temperature profile of float 2902264"*) to optimized SQLite queries using Groq (Llama-3.3-70B) with Gemini fallback support.
2. **Double-Layer Security Guard**: 
   - Restricts commands strictly to read-only `SELECT` queries.
   - Intercepts destructive SQL injection attempts (e.g. `DROP TABLE`, `DELETE`) before they are executed.
3. **Self-Correction Retry Loop**: Automatically detects SQLite database syntax errors during query execution, feeds the error back to the LLM, and regenerates a valid query on-the-fly.
4. **Static Caching Layer**: Pre-computes and caches key demo questions to load results in **~10ms**, bypassing API latencies and rate limits during presentation.
5. **Dynamic Oceanographic Visualization**: Renders interactive geographical float locations on a **Folium map** alongside vertical depth profiles (Temperature and Salinity vs depth, with reversed Y-axis) in **Plotly**.
6. **Robust Data ETL**: Seamlessly seeds the SQLite database with 975 rows of oceanographically realistic synthetic drift profiles distributed across the Arabian Sea, Bay of Bengal, and Equatorial Indian Ocean.

## 🛠️ Setup & Execution

### 1. Prerequisites & Installation
Ensure you have Python 3.10+ installed.

```bash
# Clone the repository and navigate inside
cd "ver 1"

# Install package dependencies
pip install -r requirements.txt
```

### 2. Configure Environment Keys
Create or modify the `.env` file in the root directory:
```ini
GROQ_API_KEY=your_groq_api_key_here
GEMINI_API_KEY=your_gemini_api_key_here
DB_PATH=data/argo_data.db
```

### 3. Initialize & Populate Database
Run the ETL pipeline to initialize the SQLite tables and seed the records:
```bash
python -m src.etl.fetch_argo
```

### 4. Launch FastAPI Backend
Launch the backend query translator on port `8000`:
```bash
uvicorn src.backend.main:app --host 127.0.0.1 --port 8000 --reload
```

### 5. Launch Streamlit UI
In a separate terminal, launch the interactive user interface:
```bash
streamlit run frontend/app.py
```
Open [http://localhost:8501](http://localhost:8501) in your browser.

## 💡 Recommended Presentation Queries

Use the sidebar **Quick Start Buttons** in the UI to demonstrate instant queries, or type natural variations:
1. *"Show me the temperature profile of float 2902264"* (Triggers vertical Plotly line charts)
2. *"What's the salinity in the Arabian Sea in January 2023?"* (Triggers Folium map and coordinate markers)
3. *"Compare salinity in the Arabian Sea vs Bay of Bengal"* (Triggers comparative bar charts)
4. *"Find nearest ARGO floats to lat 12, lon 65"* (Demonstrates distance calculations on coordinates)
5. *"Delete all data from the database"* (Triggers the security intercept)
